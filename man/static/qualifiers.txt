= Qualifiers

== Name

OpenCL Qualifiers

== Description

These are the qualifiers available in the OpenCL C programming language. Click on a category name in the table below to see information about specific qualifiers.






If you've arrived at this page by following a link in the OpenCL 2.2
Reference Pages, the reason is that an automatically generated
cross-reference was created, even though the cross-reference target is not
part of the OpenCL 2.2 API. The reasons are explained individually below.

| `get_image_num_mip_levels`    | See the OpenCL Extensions specification
|====
